What is RLSD?

RLSD is a lightweight, single-user operating system.


What's special about RLSD?

RLSD ships with many classic, lightweight applications, adapted to run on modern hardware.


How do I configure my wireless network card?

Go to the settings panel (through the Settings icon) and choose Wireless Wizard.


How do I install additional software?

Go to the settings panel and choose Package Manager.


How do I enable remote access to RLSD?

Either enable Dropbear through the settings panel (for command-line access) or install and run x11vnc (for graphical access).


What about security?

RLSD gives you the freedom to do whatever you want. However, this harms security, so critical system services or servers (e.g tftpd) are isolated from the rest of the system.

In addition, applications can be executed in a restricted environment (i.e so they cannot alter the system). If you mistrust an application, run it with odus, e.g odus rox.


How do I access drives?

Go to the settings panel and choose Mounting Wizard.


Where can I find more documentation?

Each and every component of RLSD is accompanied by its documentation. It is available under /usr/share/doc: for example, to see Dropbear's license, go to /usr/share/doc/dropbear.


Where can I connect with other users?

RLSD has an IRC channel: #rlsdlinux at irc.freenode.net. You can connect to it using ScrollZ or X-Chat.
